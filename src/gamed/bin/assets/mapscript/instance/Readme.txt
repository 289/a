1.副本脚本文件命名规则：instance_XXXX.lua  (XXXX代表副本模板id)

2.函数说明
-- *************************************************************** --
-- ***********           副本脚本操作函数说明           ********** --
-- *************************************************************** --
--
-- 计时器和计数器是服务器预先开好的，1至100可用，策划自行控制
--
-- 参数说明：X1-i（xxxx）表示是第一个参数，类型是整型，（备注）
--           X2-d（xxxx）表示是第二个参数，类型是浮点型，（备注）
--
--
--  ******** 地图元素相关 ********
-- （1）ActivateMapElem: 激活地图元素
--      参数：X1-i(mapid)，X2-i(地图元素id)
--
-- （2）DeactivateMapElem：取消激活地图元素、关闭地图元素
--      参数：X1-i(mapid)，X2-i(地图元素id)
--
--
--  ******** 任务相关 ********
-- （1）DeliverTask：发放任务给玩家
--      参数：X1-i(mapid)，X2-i(玩家角色id：role_id)，X3-i(任务id)
--
-- （2）DeliverTaskToAll：发放任务给副本内所有在线的玩家
--      参数：X1-i(mapid)，X2-i(任务id)
--
--
--  ******** 计数器相关 ********
-- （1）IncreaseCounter：增加计数器
--      参数：X1-i(mapid)，X2-i(计数器索引号)，X3-i(增加的值)
--
-- （2）DecreaseCounter：减少计数器
--      参数：X1-i(mapid)，X2-i(计数器索引号)，X3-i(减少的值)
--
-- （3）AssignmentCounter：给计数器赋值
--      参数：X1-i(mapid)，X2-i(计数器索引号)，X3-i(赋值的值)
--
-- （4）GetCounter：获取计数器的值
--      参数：X1-i(mapid)，X2-i(计数器索引号)
--
-- （5）LockCounter：锁定计数器，锁定后计数器值不再改变
--      参数：X1-i(mapid)，X2-i(计数器索引号)，X3-i(锁定的值)
--
-- （6）UnlockCounter：解锁计数器
--      参数：X1-i(mapid)，X2-i(计数器索引号)
--
-- （7）IsCounterLocked：该计数器是否已经锁定，返回值：0表示没有锁定，1表示已经锁定
--      参数：X1-i(mapid)，X2-i(计数器索引号)
--
--
--  ******** 计时器相关 ********
-- （1）ActivateClock：激活计时器
--      参数：X1-i(mapid)，X2-i(计时器索引号)，X3-i(重复次数)，X4-i(每次超时时间)
--
-- （2）DeactivateClock：取消计时器
--      参数：X1-i(mapid)，X2-i(计时器索引号)
--
--
--  ******** 副本控制相关 ********
-- （1）CloseInstance：关闭副本
--      参数：X1-i(mapid)，X2-i(玩家副本挑战结果：0表示玩家失败，1表示玩家胜利)
--
--
--  ******** 副本地图提示信息 ********
-- （1）ShowPromptMessage：让客户端显示提示信息
--      参数：X1-i(mapid), X2-i(显示第几句话，与客户端的xml一致), X3-i(延迟几秒), X4-i(信息显示持续几秒)
--
-- （2）ShowCountDown：让客户端显示倒计时
--      参数：X1-i(mapid), X2-i(倒计时数，从即开始倒计时)，X3-i(屏幕显示位置的x坐标)，X4-i(屏幕显示位置的y坐标)
--
--
--  ******** 地图元素移动 ********
-- （1）SpotMapElemTeleport：定点地图元素瞬移
--      参数：X1-i(mapid)，X2-i(地图元素id)，X3-d(目标点x坐标)，X4-d(目标点y坐标)，X5-i(对象瞬移后的朝向0~7)
--
-- （2）SpotMonsterMoveTo：定点怪移动到某个点
--      参数：X1-i(mapid)，X2-i(地图元素id)，X3-d(目标点x坐标)，X4-d(目标点y坐标)，X5-d(移动速度,米每秒)
--
-- （3）SetSpotMonsterSpeed：设置定点怪的移动速度
--      参数：X1-i(mapid)，X2-i(地图元素id)，X3-d(移动速度，米每秒)
--
